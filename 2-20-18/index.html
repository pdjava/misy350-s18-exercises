About MISY-350

This course will focus on current technologies and tools that are used to develop web applications in a business environment. A wide range of client-side and server-side technologies, such as HTML, CSS, JavaScript, Python, and Flask, will be covered. Tools and systems for development, testing, and deployment, such as Atom, Github, and AWS, will be introduced. This course prepares students for full-stack web development, which is one of the most popular developer occupations today.

About First Half of the Class

We will be learning about Github and the basic setup. This will include learning about different commands to properly push a file into Github website.

Further we will also learn about Hyper Text Markup Language (HTML), Cascading Style Sheets(CSS), Bootstrap, and JavaScript.
